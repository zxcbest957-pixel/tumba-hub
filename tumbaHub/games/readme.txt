TumbaHub Games Specific Profiles
================================

To add script configurations and overrides for a specific Roblox game:
1. Find the game's Place ID (e.g. 1234567890).
2. Create a file named after the Place ID inside this folder (e.g. `1234567890.lua`).
3. Write your custom Lua script inside it. It will be loaded automatically by `tumbaHub.lua` when playing that game.

For game-wide configurations that span multiple places under the same Game ID, create a file named after the Game ID instead (e.g. `2345678901.lua`).
